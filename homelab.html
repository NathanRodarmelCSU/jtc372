Home Lab & Virtualization

My home lab is built around Proxmox and runs multiple virtual machines that I use for both day-to-day tasks and system management. It serves as a small, self-managed environment where I run services, maintain systems, and troubleshoot issues as they come up.

Proxmox Virtual Enviroment

Proxmox is the core of my environment, hosting and managing all of my virtual machines and containers.

TrueNAS Storage

TrueNAS provides centralized storage for my environment, handling file sharing, backups, and data organization.

Windows Virtual Machines

I use Windows VMs for everyday tasks as well as IT-related work such as configuration, troubleshooting, and remote access. Some systems are also used to test changes before applying them elsewhere.

Linux Server VMs

My Linux VMs handle both regular use and server-based workloads. I use them for running services, managing systems through the command line, and maintaining a stable environment, while keeping separate machines available for testing.

ADS-B / FR24 Server

Docker & Network Services
